# 🎮 拯救Zack - MT冒险游戏

一个基于HTML5的2D冒险解谜游戏，玩家需要在Magic Tavern公司的各个区域中穿越重重障碍，收集钥匙，最终拯救被困的CEO Zack。

## 🌟 游戏特色

### 🎯 核心玩法
- **10个精心设计的关卡**：从公司前台到CEO办公室的完整冒险旅程
- **渐进式难度**：从新手友好到终极挑战的平滑难度曲线
- **钥匙收集系统**：第3-5关需要收集3把钥匙才能通关
- **多样化陷阱系统**：静态陷阱、移动陷阱、特殊陷阱等15+种不同类型

### 🎨 视觉效果
- **响应式设计**：完美适配桌面和移动设备
- **地狱模式视觉**：第10关特殊的红色主题和动画效果
- **流畅动画**：移动陷阱的浮动效果和玩家移动动画
- **直观UI**：清晰的进度显示和关卡信息

### 🎵 音效系统
- **背景音乐**：沉浸式的游戏音乐体验
- **音效反馈**：移动、收集、爆炸等多种音效
- **音频控制**：独立的音乐和音效开关

## 🎮 操作方式

### 💻 桌面端
- **方向键**：↑↓←→ 控制角色移动
- **鼠标点击**：操作游戏界面按钮

### 📱 移动端
- **拖拽控制**：直接拖拽角色进行移动
- **虚拟方向键**：屏幕右下角的备用控制方案
- **触摸收集**：点击钥匙进行收集

## 🎯 陷阱系统

### 📍 静态陷阱
- **方块陷阱**：基础的矩形障碍物

- **智能移动**：具有边界反弹的动态障碍
- **视觉升级**：使用隐形陷阱图标和浮动动画

5. **🌀 重力陷阱**：将玩家拉向中心
6. **🏀 弹跳陷阱**：将玩家弹向随机方向
7. **🔍 缩小陷阱**：让玩家变小5秒
8. **⚡ 加速陷阱**：提升移动速度5秒
9. **👥 克隆陷阱**：创建玩家分身
10. **🎯 追踪陷阱**：主动追踪玩家
11. **💨 闪烁陷阱**：间歇性显示/隐藏
12. **🔮 镜像陷阱**：反转控制方向

## 🚀 快速开始

### 📋 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持HTML5和CSS3
- 启用JavaScript

### 🎯 运行游戏

#### 方法一：直接打开
```bash
# 直接在浏览器中打开 index.html 文件
open index.html
```

#### 方法二：本地服务器
```bash
# 使用Python启动本地服务器
python3 -m http.server 8001

# 或使用Node.js
npx http-server -p 8001

# 然后在浏览器中访问
http://localhost:8001
```

### 🎮 开始游戏
1. 打开游戏页面
2. 点击开始或直接使用方向键移动
3. 避开红色陷阱，到达绿色出口
4. 在第3-5关收集所有钥匙
5. 挑战第10关的地狱模式！
